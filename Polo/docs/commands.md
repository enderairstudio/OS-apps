# Polo Commands

`say <text>` prints text back to the terminal.

`cd <path>` changes the active tab directory.

`pwd` prints the active tab directory.

`clear` or `cls` clears the terminal output.

`clip3d` or `3d` clears the terminal surface and shows the colored 3D clip. Press `Esc` to return.

`exit` closes the active tab. Closing the last tab quits Polo.

## Colors

The first command word is highlighted yellow while typing.

Invalid commands and built-in errors are red.

External applications can print ANSI color escapes and Polo will render the foreground color.
